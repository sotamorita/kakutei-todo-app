# 確定申告ナビ

<img width="2816" height="1536" alt="Image" src="https://github.com/user-attachments/assets/fb351c2b-0788-49c3-8944-33028351fd85" />

確定申告が必要かどうかを診断し、必要な書類や手続きをガイドするWebアプリケーションです。

## 使い方

1. 質問に順番に答えていく
2. 該当する項目について「はい」を選択
3. 診断結果とToDoリストを確認
4. 必要に応じてAIアシスタントに質問

## 主な機能

- 確定申告の必要性診断
- 必要書類と手続きのリスト化
- 国税庁公式サイトへのリンク
- 用語解説
